只有APK檔也能自動化測試！？Appium介紹及安裝

2020年10月29日
| Appium

如果你老闆或PM開出以下需求：

1、一台電腦控制多裝置(實機或模擬器)

2、能以腳本控制執行測試

3、甚至要在只有apk檔的情況下進行測試

只有滿足上任一項目，你就可以考慮使用Appium來自動化測試。

Whaz Appium

Appium 是一套開源的自動測試工具，可以用來測試原生(Native)、混合(Hybrid)、mobile web 三種行動應用開發。
